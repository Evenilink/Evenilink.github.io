Ganbatte

Ganbatte is a competitive social multiplayer arcade game for VR, about cats, in space, eating sushi. Try to eat as much valuable sushi as you can and reach the highest score.

  • Gameplay implementation, such as the plate’s respawn system, combo system, player interaction and game state.
  • Worked on all of the networking aspects of the game.
  • Used Behavior Trees to program bot’s AI (and adapt it to a multiplayer setting).
  • Programmed a lot of the user interface.
  • Worked a lot with the Persona System in order to implement animation controllers
  • Worked constantly with artists and designers to build the levels, adapting all the 3D models and the animations.
  • Very active on the game’s design process through ideas, concepts and game balancing.

Skills

3D Math

Had to program numerous gameplay features that required mathematical concepts, like enemy perception (vision and hearing), hands slapping detection and a stealth cover system.

Artificial Intelligence

This is one of my favorite fields. I’ve worked with Finite State Machines and Behavior Trees, implemented features like A* pathfinding and Dijkstra shortest path, a multi-agent cooperative system in which the agents have to exit an unknown space, and even my own Behavior Tree System for Unity, all of these from scratch.

Networking

I’ve used techniques like client-side prediction, server reconciliation and entity interpolation, as well as working with replication models.
