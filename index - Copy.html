Ganbatte

Ganbatte is a competitive social multiplayer arcade game for VR, about cats, in space, eating sushi. Try to eat as much valuable sushi as you can and reach the highest score.

  • Gameplay implementation, such as the plate's respawn system, combo system, player interaction and game state.
  • Worked on all of the networking aspects of the game.
  • Used Behavior Trees to program NPCs' AI (and adapt it to a multiplayer setting).
  • Programmed a lot of the user interface.
  • Worked a lot with the Persona System in order to implement animation controllers.
  • Worked constantly with artists and designers to build the levels, adapting all the 3D models and the animations.
  • Very active on the game's design process through ideas, concepts and game balancing.

Personal Projects

Magis

A VR game where the player is a wizard and can shapeshift into other creatures with a variety of abilities, enabling them to solve puzzles. Be a cat to see in the dark, a crow to fly to the skies, a monkey to climb what's in your way, and a hamster to enter tiny holes on the walls!

  • Implemented each creature's gameplay (like running using the touch controllers, grabbing objects, shapeshift into other creatures), inventory system and scripted events.
  • Used Behavior Trees to program the game's AI.
  • Animation Controllers implementation.
  • Handled all the sound programming.
  • Worked with a team of programmers, artists and sound designers (10 people).

Deus Ex based Stealth Prototype

A "Deus Ex: Human Revolution" inspired stealth prototype.

  • Pawn sensing (vision and hearing) detection system for the AI enemies.
  • Implemented cover system, allowing the character to move around corners and to jump between cover to cover.
  • Grabbable objects that are grabbed from the side the player is facing.
  • Stealth radar that takes player and enemy rotation into account.

Behavior Tree Plugin for Unity

My own Behavior Tree plugin for the Unity Engine, complete with Blackboards, Composite Nodes, Tasks, Decorators and Behavior Trees. The project repository is available here.

Unstatic

A Star Wars inspired multiplayer-droid-racing game (each droid with a unique special ability).

  • Gameplay implementation, such as the player controller, lap control and character's abilities.

Skills

3D Math

Had to program numerous gameplay features that required mathematical concepts, like enemy perception (vision and hearing), hands slapping detection and a stealth cover system.

Artificial Intelligence

Networking

I’ve used techniques like client-side prediction, server reconciliation and entity interpolation, as well as working with replication models.
